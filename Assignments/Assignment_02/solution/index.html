Assignment 2

In this assignment you will apply following web technologies:

  1. Front-end / client:
    • dynamic HTML-pages by means of javascript
    • event-driven execution of javascript-functions
    • using simplifications offered by the jQuery-library
    • asynchronous communication with a server
    • DOM manipulation
  2. Back-end / server:
    • basic communication via sockets
    • communicating via the HTTP-protocol
